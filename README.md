Kurulum için

- mapbox.com hesabı oluşturup api token alın
- Token'i aşağıda belirtilen şekilde ekleyin `/.env.local`

```
REACT_APP_MAPBOX_ACCESS_TOKEN=token
```

- `npm i && npm start`
